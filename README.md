# Sintaxe Visual & Arquétipos Humanos

## 🇺🇸 English

### Overview
This project is a web application built with **React**, **TypeScript**, and **Vite**. It explores visual syntax and human archetypes, offering a rich and interactive user experience.

### Technologies Used
- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion**
- **React Router DOM**
- **Lucide React** (Icons)

### Getting Started

To run this project locally, follow these steps:

1. **Clone the repository** (if applicable) or download the files.
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
